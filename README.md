# step-bunching-program

Repository centered on **step bunching as a physical phenomenon** across multiple methods.

## Scope

This repository is focused on the broader step-bunching domain, including:

- ODE-based models of step dynamics
- scaling and dimensional analysis
- experiment–theory comparison
- classification schemes
- generalized bunching models
- legacy codes and modern reimplementations

The separate `CA` repository is centered on **Cellular Automata as a method** and therefore includes different physical objects, not only vicinal-surface problems.

## Organizing idea

This repository is **phenomenon-centered** rather than method-centered.

Its purpose is to collect, organize, and further develop the non-CA and multi-method line of the step-bunching research program, especially the historical domain developed roughly from 1997 to 2015, together with future extensions and modernizations.

## Main sections

- `docs/` — conceptual and historical framework
- `bibliography/` — core literature and citation maps
- `legacy_code/` — older codes and reproducibility notes
- `src/` — modern implementations and analysis tools
- `examples/` — minimal working examples
- `data_notes/` — links to experimental and benchmark contexts
- `notes/` — internal programmatic notes
