# B2-type overview

## General idea

This file is devoted to step-bunching systems and models of **B2-type**.

## Defining property

In B2-type bunching, the minimal step-step distance inside the bunch decreases as the bunch size increases.

## Main implication

B2-type bunching is characterized by **two characteristic length scales**.
