# B1-type overview

## General idea

This file is devoted to step-bunching systems and models of **B1-type**.

## Defining property

In B1-type bunching, the minimal step-step distance inside the bunch remains approximately constant as the bunch size increases.

## Main implication

B1-type bunching is characterized by **one characteristic length scale**.
