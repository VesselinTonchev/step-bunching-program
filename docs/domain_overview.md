
# Domain overview

## General idea

This repository is centered on **step bunching** as a scientific domain.

Unlike the `CA` repository, which is method-centered, the present repository is organized around a single physical phenomenon studied through multiple approaches.

## Covered lines

The repository is intended to include:

- ODE-based models of step motion
- scaling and dimensional-analysis approaches
- experiment–theory comparison
- classification schemes
- generalized bunching models
- legacy computational tools and modern reimplementations

## Historical emphasis

A large part of the historical core of this repository belongs to the period roughly between **1997 and 2015**, when the step-bunching program was developed mainly outside the Cellular Automaton framework.

## Relation to the CA repository

The CA-based branch of step bunching belongs methodologically to the separate `CA` repository.

The present repository may still refer to that branch when needed, but its main purpose is to preserve and develop the broader multi-method step-bunching line.
